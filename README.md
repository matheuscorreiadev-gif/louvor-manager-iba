# Sistema de Escalas - Ministério de Louvor
Igreja Batista Aliança

Projeto desenvolvido como parte do Projeto de Extensão (PEX)
do curso de Análise e Desenvolvimento de Sistemas.

## Objetivo
Organizar e estruturar as escalas mensais do Ministério de Louvor,
substituindo o modelo atual realizado via WhatsApp.

## Tecnologias Utilizadas
- HTML
- CSS
- JavaScript
- LocalStorage
- GitHub Pages

## Funcionalidades
- Cadastro de membros
- Organização por mês
- Separação por tipos de culto
- Sistema de login simples
- Persistência de dados no navegador
